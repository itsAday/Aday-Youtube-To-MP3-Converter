# Aday Youtube To MP3 Converter
Simple Python Youtube to MP3 Converter made with ♡ by me

# Instructions:
- Download and extract latest release
- Execute "AdayYoutubeConversor.py"
- Wait for libraries to install
- Paste Youtube URL
- Done :D


Have fun! This is way better than all of the online websites with hundreds of annoying ads.
